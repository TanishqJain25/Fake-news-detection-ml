# Fake News Detection using Machine Learning

This project implements a Fake News Detection system using Natural Language Processing techniques and Machine Learning models to classify news articles as Fake or True.

## Overview

The objective of this project is to build a text classification model using TF-IDF vectorization and supervised learning algorithms. The system is trained on labeled news datasets and evaluated using standard classification metrics.

## Technologies Used

Python  
Pandas  
NumPy  
Scikit-learn  
Matplotlib  

## Machine Learning Pipeline

1. Data Loading from True and Fake datasets  
2. Data Cleaning and Preprocessing  
3. TF-IDF Feature Extraction  
4. Train-Test Split  
5. Model Training  
   - Logistic Regression  
   - Multinomial Naive Bayes  
6. Model Evaluation  

## Evaluation Metrics

Accuracy Score  
Confusion Matrix  
Precision  
Recall  
F1-Score  

Logistic Regression achieved higher accuracy compared to Naive Bayes on the given dataset.
